# Introduction

WildSync is a social platform designed to connect outdoor enthusiasts. It allows users to plan activities, discover new adventures, and meet like-minded people who share a passion for the outdoors.

## Key Features

- **User Management**: Allow user to register, login, and manage their personal profile, including basic information, avatar photo, and personal page banner image.
- **Activities List**: Browse and search for outdoor activities based on event difficulty, time, and location, with pagination for easy navigation.
- **Activity Page**: Display detailed information about each activity, allowing holder to create, edit, and delete and participants to like, join or leave activities.
- **Personal Page**: Showcase user's personal information and avatar, listing out their created and joined activities for others to view.
- **Browse History & Like**: Keep track of recently viewed and liked activities to assist users in finding activities they are interested in.

## Technical Insights

- **Frontend**: Built with React for a dynamic and responsive user interface, allowing components to be reused and managed efficiently.
- **Database**: Utilized Firebase Firestore for real-time data storage and retrieval, ensuring seamless synchronization across devices.
- **State Management**: Implemented Redux for efficient state management, enabling predictable state transitions and maintaining single source of truth.

## Challenges and Solutions

- **State Management**: Overcame challenges in managing complex state interactions by leveraging Redux, ensuring a consistent and predictable application state.
- **Data Fetching**: Optimized data fetching strategies to reduce load times and improve user experience, especially for activities list with pagination.
- **Deployment**: Successfully deployed the application on Github Page, ensuring reliable access and performance.

## Outcome

WildSync demonstrates effective use of React, Firebase, and Redux to create a seamless user experience.
